# Iconos del Portfolio

Esta carpeta contiene los iconos y favicons del sitio web.

## Archivos recomendados:

### Favicon básico:
- `favicon.ico` - Favicon tradicional (16x16, 32x32, 48x48)
- `favicon.svg` - Favicon vectorial moderno

### Iconos para diferentes dispositivos:
- `apple-touch-icon.png` - Icono para iOS (180x180)
- `icon-192.png` - Icono PWA pequeño (192x192)
- `icon-512.png` - Icono PWA grande (512x512)

### Iconos adicionales:
- `favicon-16x16.png` - Favicon 16x16
- `favicon-32x32.png` - Favicon 32x32

## Uso:
Los archivos en esta carpeta se copiarán automáticamente a la raíz del sitio durante el build.

## Formatos recomendados:
- **ICO**: Para máxima compatibilidad
- **SVG**: Para calidad vectorial
- **PNG**: Para iconos específicos de tamaño
